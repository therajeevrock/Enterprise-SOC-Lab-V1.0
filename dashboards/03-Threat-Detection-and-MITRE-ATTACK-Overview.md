# Create Dashboard

## Objective

Create a new dashboard that will contain visualizations related to threat detections, alert severity, affected endpoints, and MITRE ATT&CK mappings. This dashboard provides SOC analysts with a centralized view for monitoring security detections and understanding attacker behavior.

---

## Step 1 - Open the Dashboards Page

From the Wazuh Dashboard navigation menu:

```text
Menu
→ Dashboards
```

Click **Create dashboard**.

### Screenshot

**Image #01**

```text
dashboards/screenshots/dashboard-03/01-open-create-dashboard.png
```

---

## Step 2 - Configure Dashboard Details

Provide the following information.

### Title

```text
Enterprise SOC - Threat Detection & MITRE ATT&CK Overview
```

### Description

```text
The Threat Detection & MITRE ATT&CK Overview dashboard provides SOC analysts with a centralized view of security detections, alert severity, affected endpoints, and adversary techniques mapped to the MITRE ATT&CK framework. It enables analysts to prioritize threats, identify attack patterns, understand attacker behavior, and improve incident investigation and response through actionable security insights.
```

Enable the following option:

```text
Store time with dashboard
```

Click **Save** to create the dashboard.
